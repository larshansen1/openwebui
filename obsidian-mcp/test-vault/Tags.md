# Working with Tags

Tags help organize your notes in the vault.

## Tag Syntax

You can add tags in two ways:

1. **Frontmatter tags** (in YAML header above)
2. **Inline tags** using `#tagname` syntax

## Examples

This note has frontmatter tags: `tags`, `organization`

You can also use inline tags like #example #testing #mcp-server

## Searching by Tags

Use the MCP server's `search_notes` tool with tag filters to find all notes with specific tags.

See [[API Documentation]] for details.
