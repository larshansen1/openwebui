# Quick Start Guide

Get started with the Obsidian MCP Server in minutes.

## Installation

1. Configure your vault path in `.env`
2. Start the service with `docker compose up`
3. Access the [[API Documentation]]

## Basic Usage

### Searching Notes

Use the search functionality to find notes by content or [[Tags]].

### Creating Notes

You can create new notes with frontmatter automatically added.

### Wiki Links

Link to other notes like [[Welcome]] or [[File Watcher]] using wiki-link syntax.
